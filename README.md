# Spreadsheet Health Audit Checklist

A practical checklist for identifying structural risks in operational spreadsheets.

Many businesses rely heavily on spreadsheets for reporting, planning, and operational workflows.  
Over time those files often become fragile, difficult to maintain, and risky to rely on.

This checklist helps identify common structural issues before they create problems.

---

## Structural risks to check

- formulas referencing inconsistent ranges
- calculations mixed with raw data
- manual copy-paste workflows
- hidden logic across multiple sheets
- lack of validation rules
- multiple file versions circulating

---

## Signs a spreadsheet has become operationally fragile

- only one person understands the file
- small changes cause unexpected results
- reports must be rebuilt manually each week
- errors appear when new rows are added

---

## Recommended structural principles

- separate inputs, calculations, and outputs
- minimise nested formulas
- document logic assumptions
- build consistent refresh workflows

---

## When a deeper review helps

If the spreadsheet supports important operational decisions, a structured audit can help identify risks early.
